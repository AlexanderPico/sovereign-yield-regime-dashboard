# Sovereign Yield Regime Dashboard

- Incubation date: 2026-05-14
- Source brief filename: `sovereign-yield-regime-dashboard.md`
- Why promoted now: this was the strongest top-level project brief with both high maturity and a bounded first milestone (`Week 1 prototype`) that is ready to become an active software workspace.

## Current thesis

This project should become a local, inspectable sovereign-yield regime interpreter rather than another chart-only macro dashboard. The first useful artifact is a small system that ingests reliable public yield data, computes curve and spread features, and emits auditable regime labels with brief narrative explanations.

## First milestone

Build a week-1 prototype that pulls U.S. Treasury/FRED data, computes basic slope and change features, and labels a small set of transparent regimes.

## Immediate next steps

1. Confirm the first-country dataset contract for U.S. Treasury and FRED series.
2. Define the initial regime taxonomy and explicit label thresholds.
3. Sketch the minimal data pipeline and local project structure.
4. Build a notebook or small app that computes 10Y-2Y and 10Y-3M features.
5. Generate one sample regime card with evidence, contradictions, and confidence.
